# ReelIQ — Instagram Reel Diagnostic Tool

AI-powered content performance diagnostic system for early-stage Instagram creators and SMEs.

## What it does
ReelIQ analyses Instagram Reel metrics and generates plain-language diagnostic reports explaining why content underperformed and what to change next.

## Built with
- Python + Streamlit
- OpenAI GPT-4o mini
- Custom diagnostic scoring engine
